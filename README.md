# Expenses Tracker App
Expenses tracker app to manage your money! 💰
## Technologies used
* ReactJS
* Typescript
* Styled components
* MirageJS
## How to Run
```
# Clone Repository
$ git clone https://github.com/danielbeckera/expenses-tracker-enhanced.git
```
```
# Install Dependencies
$ npm i

# Run Aplication
$ npm start
```
Visit http://localhost:3000/ to see the app 👩‍💻
